In this page we are comparing different image formats. 1. GIF 2. JPG 3. PNG 4. WebP




Below you can see the list of photos and their size. List is shown in png (95kb) and in WebP (32kb) formats (again for comparison).


The photos seems to be of similar quality, it is difficult to tell which are better. I may need to adjust this if I find quality differences.
From the tables you can see that gif and png types are a lot heavier, they take longer to load.
jpg and WebP load much faster. We should look to use these image file types in websites. Lean towards WebP.
